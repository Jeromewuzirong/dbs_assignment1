DBS Assignment 1 — Design Gallery

25 Versions of a PC Game Guide

A landing page for PC game recommendations, evolved through 25 design iterations — from raw exploration to polished convergence.

Phase 1 Exploration Wildly different aesthetics, testing boundaries
V1 Origin
Ultra-Minimal Serif
Pure white background, elegant Georgia typography, maximum whitespace. Top 5 games in a clean numbered list. The quietest possible starting point.
View
V2 New Style
Dark Cyberpunk
Neon green and purple on black. Scanline overlay, grid background, Orbitron font. Cards with purple left-border that flips green on hover. The cyberpunk DNA begins.
View
V3 New Style
Retro Arcade
Press Start 2P pixel font, bold red/yellow/blue blocks, blinking title, scrolling marquee, arcade button-press hover effect. All 8 games debut here.
View
V4 New Style
Editorial Magazine
Playfair Display serif headlines, Source Sans body, warm cream background, full-bleed hero gradient, pull quote divider. Bon Appetit meets gaming.
View
V5 New Style
Brutalist Anti-Design
Black and white only. 6px borders, Anton condensed font, chaotic 3-column grid with misaligned cards, manifesto text. Full card color inversion on hover.
View
V6 New Style
Y2K Geocities
Rainbow gradients, chrome text, blinking everything, sparkle cursors, visitor counter, guestbook, webring. Maximum early-internet chaos energy.
View
V7 New Style
Console Dashboard
PS5-inspired dark UI. Inter font, horizontal scroll rail with snap, hero spotlight card, detail list rows. Sleek, modern, system-level feel.
View
V8 New Style
Hand-Drawn Zine
Caveat handwriting font, photocopy grain texture, tilted cards, tape strips, marker highlights, sticky notes, doodle borders. Photocopied with love.
View
V9 New Style
Hacker Terminal
Fira Code monospace, green-on-black, fake CLI boot sequence, ASCII art title, SQL query output, progress bars, blinking cursor. CRT vignette effect.
View
V10 New Style
Luxury Gold
Cormorant Garamond serif, deep charcoal background, gold champagne accents, thin dividers, diamond ornaments. Luxury watch brand aesthetic.
View
Phase 2 Fusion Combining the strongest ideas from Phase 1
V11 V2 + V3
Cyberpunk Pixel Fusion
V2's neon cyberpunk palette with V3's pixel font. Press Start 2P headings, Share Tech Mono body. The core aesthetic identity is born.
View
V12 V2 + Pixel Art
Pixel-Art Borders
V2's cyberpunk layout with chunky 4px stepped borders using layered box-shadows. L-shaped corner brackets pulse on each card. Pixel meets neon.
View
V13 8-Bit + Neon
8-Bit Neon Grid
Pixel font for everything, 8 bold accent colors cycling per card, 2-column chunky grid, rainbow color bars, hearts/lives bar, neon glow on titles.
View
V14 V2 + Data Viz
Pixel Progress Bars
Cyberpunk dark theme with all ratings replaced by block-character progress bars. S/A/B tier color coding in green, cyan, and amber.
View
V15 V11 Recolor
Synthwave Pink
V11's layout recolored from green/purple to hot pink/deep purple. Subtle purple gradient wash on the background. Vaporwave meets retro.
View
V16 V13 + Carousel
Pixel Carousel
V13's 8-bit neon style converted to horizontal scrolling cards with scroll-snap, arrow navigation buttons, and a live card counter.
View
Phase 3 Animation & Interaction Bringing the fused designs to life
V17 V12 + Motion
Animated Pixelgrid
V12 base with 6 animation layers: blinking cursor, neon voltage flicker, pixel button-press, floating featured card, pulsing corners, shimmering lines.
View
V18 Refined
Typography Refinement
Best of v11-v17 unified with polished typography hierarchy: larger titles, clear section headings, 2-column rank+content layout, generous whitespace.
View
V19 Pre-Final
Synthgrid — Pre-Final
The unified vision: v15's pink/purple palette, v17's animations, v18's spacing. Navigation, featured card, summary table. Feels complete and shippable.
View
Phase 4 Cover Art & Reveal Adding real game images and interactive reveals
V20 Images
Cover Art Edition
V17 base with Steam header images on every card. Featured card gets full-width hero banner; regular cards use side-by-side image+content grid layout.
View
V21 Interaction
Hover Reveal
Cards start compact (title + score only). On hover, cover image and description expand smoothly via CSS max-height transition. Pure CSS, no JavaScript.
View
V22 V21 Softened
Muted Palette
V21 with softened colors: neon green becomes muted teal, purple becomes lavender. Lower glow radii, gentler flicker. A more refined take.
View
V23 V22 + V17 Colors
Neon Restored
Reverts to original v17 neon green/purple palette while keeping v21's hover reveals. Card expand transitions slowed to 0.6s for a deliberate feel.
View
V24 Enhancement
Background Peek + Big Ranks
Each card shows a faint ghost of the cover image (opacity 0.12) before hover. Ranking numbers become 3.2rem editorial-style numbers on the left.
View

Final — Top PC Games
